# Hyper Cmd Lib Net

Network Library to Interface with Hyperswarm and local connections. Supports both UDP and TCP connections.

## Install

`npm i @holesail/hyper-cmd-lib-net`

## API

#### `connPiper`

#### `udpPiper`

#### `udpConnect`

#### `createTcpProxy`

#### `pipeTcpServer`

#### `createUdpFramedProxy`

#### `pipeUdpFramedServer`
